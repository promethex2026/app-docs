## Official SDKs

PrometheX provides client libraries to simplify integration. SDKs handle authentication, request signing, UserOperation construction, and type safety.

### JavaScript / TypeScript

<Card title="@promethex/sdk" icon="js">
  TypeScript-first SDK with full type definitions. Works in Node.js and browsers.

```bash
npm install @promethex/sdk
```

```typescript
import { PrometheX } from '@promethex/sdk';

const client = new PrometheX({
  baseUrl: 'https://api-staging.promethex.market',
  token: privyJwt,
});

// List markets
const markets = await client.markets.list({ page: 1, pageSize: 10, sortType: 'hot' });

// Get market detail
const market = await client.markets.get('0x1234...abcd');

// Place a buy order
const order = await client.trading.buy({
  marketAddress: '0x1234...abcd',
  optionAddress: '0xaaaa...1111',
  amount: '10000000',
  slippage: 0.05, // 5% slippage tolerance
});
```

## React SDK

For frontend applications, the React SDK provides hooks and components:

{/* TODO: Publish React SDK package */}

```bash
npm install @promethex/react-sdk
```

```tsx
import { PrometheXProvider, useMarkets, useTrading } from '@promethex/react-sdk';

function App() {
  return (
    <PrometheXProvider
      apiUrl="https://api-staging.promethex.market"
      privyAppId="your-privy-app-id"
    >
      <MarketList />
    </PrometheXProvider>
  );
}

function MarketList() {
  const { data: markets, isLoading } = useMarkets({ sortType: 'hot' });
  const { buy } = useTrading();

  if (isLoading) return <div>Loading...</div>;

  return markets.map(market => (
    <div key={market.marketAddress}>
      <h3>{market.title}</h3>
      <button onClick={() => buy({
        marketAddress: market.marketAddress,
        optionAddress: market.options[0].optionAddress,
        amount: '10000000',
      })}>
        Buy Yes
      </button>
    </div>
  ));
}
```

## REST API Direct

If no SDK is available for your language, use the REST API directly. All endpoints accept JSON and return the standard response envelope.

<CodeGroup>

```bash cURL
curl "https://api-staging.promethex.market/api/v1/market/get-markets?page=1&pageSize=10" \
  -H "Content-Type: application/json"
```

```go Go
package main

import (
    "encoding/json"
    "net/http"
)

func main() {
    req, _ := http.NewRequest("GET",
        "https://api-staging.promethex.market/api/v1/market/get-markets?page=1&pageSize=10",
        nil,
    )
    req.Header.Set("Authorization", "Bearer "+privyJwt)
    resp, _ := http.DefaultClient.Do(req)

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
}
```

```rust Rust
use reqwest;

#[tokio::main]
async fn main() -> Result<(), reqwest::Error> {
    let client = reqwest::Client::new();
    let res = client
        .get("https://api-staging.promethex.market/api/v1/market/get-markets")
        .query(&[("page", "1"), ("pageSize", "10")])
        .header("Authorization", format!("Bearer {}", privy_jwt))
        .send()
        .await?;

    let body: serde_json::Value = res.json().await?;
    println!("{:?}", body);
    Ok(())
}
```
