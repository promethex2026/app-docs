---
title: Update Profile
description: "Update user avatar, name, and description"
---

## Set Avatar

```
POST /bayes/user/set-avatar
```

**Auth required:** Yes

```json Request
{ "avatarUrl": "https://..." }
```

---

## Set Name

```
POST /bayes/user/set-name
```

**Auth required:** Yes

```json Request
{
  "name": "alice",
  "inviteByCode": "XYZ789"
}
```

---

## Set Description

```
POST /bayes/user/set-description
```

**Auth required:** Yes

```json Request
{ "description": "Trading on everything" }
```

---

## Follow / Unfollow User

```
POST /bayes/user/follow-user
```

```json Request
{ "followUid": "67890" }
```

```
POST /bayes/user/unfollow-user
```

```json Request
{ "unfollowUid": "67890" }
```

All three endpoints return a standard success response:

```json
{ "code": 0, "msg": "success" }
```
