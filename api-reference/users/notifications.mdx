## Get Notifications

```
GET /bayes/user/get-user-notifications
```

**Auth required:** Yes

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `page` | number | Yes | Page number |
| `pageSize` | number | Yes | Items per page |
| `category` | enum | No | Filter by category |
| `status` | enum | No | `read` or `unread` |
| `type` | enum | No | Notification type |

### Response

```json
{
  "code": 0,
  "data": {
    "total": 23,
    "notifications": [
      {
        "uuid": "notif-123",
        "type": "trade_filled",
        "title": "Order Filled",
        "content": "Your buy order for 'Yes' on...",
        "status": "unread",
        "createdAt": "2026-02-15T10:30:00Z"
      }
    ]
  }
}
```

## Mark as Read

```
POST /bayes/user/mark-notifications-as-read
```

**Auth required:** Yes

```json Request
{
  "notificationUuids": ["notif-123", "notif-456"]
}
```
