## Get User Positions

```
GET /bayes/market/get-user-positions
```

**Auth required:** No (but `uid` needed)

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `page` | number | Yes | Page number |
| `pageSize` | number | Yes | Items per page |
| `uid` | string | No | User ID (defaults to authenticated user) |
| `marketAddress` | string | No | Filter by market |

### Response

```json
{
  "code": 0,
  "data": {
    "total": 8,
    "positions": [
      {
        "marketAddress": "0x...",
        "marketTitle": "Will BTC hit $100k?",
        "optionTitle": "Yes",
        "amount": "15000000",
        "avgPrice": "0.62",
        "currentPrice": "0.65",
        "pnl": "450000",
        "pnlPercent": "4.8"
      }
    ]
  }
}
```

## Get Portfolio Summary

```
GET /bayes/market/get-user-asset-info
```

**Auth required:** Yes

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `uid` | string | Yes | User ID |
| `baseTokenAddress` | string | Yes | Base token address |

### Response

```json
{
  "code": 0,
  "data": {
    "value": "125000000",
    "balance": "50000000",
    "portfolio": "75000000",
    "pnl": "12500000",
    "decimal": 6,
    "volume": "340000000",
    "pnlRank": 42
  }
}
```

## Get Asset History

```
GET /bayes/user/get-user-asset-history
```

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `uid` | string | Yes | User ID |
| `timeInterval` | enum | Yes | `1d`, `1w`, `1m`, `3m`, `all` |
| `baseTokenAddress` | string | Yes | Base token address |

Returns time-series data for rendering portfolio value charts.
