## Get Comments

```
GET /bayes/community/get-comments
```

**Auth required:** No

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `postUuid` | string | Yes | Parent post UUID |
| `page` | number | No | Page number |
| `pageSize` | number | No | Items per page |
| `rootUuid` | string | No | Root comment (for nested replies) |

### Response

```json
{
  "code": 0,
  "data": {
    "total": "8",
    "comments": [
      {
        "commentUuid": "comment-789",
        "content": "Great analysis! I agree.",
        "likesCount": 3,
        "isLiked": false,
        "repliesCount": 1,
        "user": {
          "uid": "67890",
          "name": "bob",
          "avatarUrl": "https://..."
        },
        "createdAt": "2026-02-15T09:00:00Z"
      }
    ]
  }
}
```

## Publish Comment

```
POST /bayes/community/publish-comment
```

**Auth required:** Yes

### Reply to a post

```json
{
  "postUuid": "post-123",
  "content": "I disagree, here's why..."
}
```

### Reply to a comment (nested)

```json
{
  "postUuid": "post-123",
  "rootUuid": "comment-789",
  "parentUuid": "comment-789",
  "content": "Good point!"
}
```

### Response

```json
{
  "code": 0,
  "data": { "commentUuid": "comment-101" }
}
```
