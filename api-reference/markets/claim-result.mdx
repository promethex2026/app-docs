---
title: Claim Result
description: "Claim winnings after a market settles"
---

## Endpoint

```
POST /bayes/market/claim-market-result
```

**Auth required:** Yes

## Request Body

```json
{
  "marketAddress": "0x1234...abcd",
  "optionAddress": "0xaaaa...1111",
  "userOperation": {
    "sender": "0x...",
    "nonce": "0x...",
    "callData": "0x...",
    "signature": "0x..."
  }
}
```

| Field | Type | Description |
|-------|------|-------------|
| `marketAddress` | string | Settled market address |
| `optionAddress` | string | Winning option token address |
| `userOperation` | object | Signed ERC-4337 UserOperation |

## Response

```json
{
  "code": 0,
  "data": {
    "opHash": "0x..."
  }
}
```

The claim transaction burns the user's winning option tokens and transfers base tokens (USDC) to their smart account at a 1:1 ratio.

<Info>
Only callable after a market has settled (status = `ended`). Users with losing option tokens cannot claim — those tokens are worthless.
</Info>
