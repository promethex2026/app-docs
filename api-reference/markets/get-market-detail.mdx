---
title: Get Market Detail
description: "Get full details for a single prediction market"
---

## Endpoint

```
GET /bayes/market/get-market-detail
```

**Auth required:** No

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `marketAddress` | string | Yes | On-chain market contract address |
| `baseTokenAddress` | string | No | Base token filter |
