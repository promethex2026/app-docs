---
title: Get Hot Markets
description: "Get trending prediction markets"
---

## Endpoint

```
GET /bayes/market/get-hot-markets
```

**Auth required:** No

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `baseTokenAddress` | string | No | Filter by base token |

## Response

```json
{
  "code": 0,
  "data": {
    "marketList": [
      {
        "marketAddress": "0x...",
        "title": "Will ETH flip BTC by 2027?",
        "status": "running",
        "volume": "89000000000",
        "options": [
          { "title": "Yes", "price": "0.12" },
          { "title": "No", "price": "0.88" }
        ]
      }
    ]
  }
}
```
