---
title: Follow / Unfollow Market
description: "Add or remove markets from your watchlist"
---

## Follow Market

```
POST /bayes/market/follow-market
```

**Auth required:** Yes

### Request Body

```json
{
  "marketAddress": "0x..."
}
```

### Response

```json
{
  "code": 0,
  "msg": "success"
}
```

---

## Unfollow Market

```
POST /bayes/market/unfollow-market
```

**Auth required:** Yes

### Request Body

```json
{
  "marketAddress": "0x..."
}
```

---

## Get Followed Markets

```
GET /bayes/market/get-follow-markets
```

**Auth required:** Yes

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `page` | number | No | Page number |
| `pageSize` | number | No | Items per page |

### Response

```json
{
  "code": 0,
  "data": {
    "total": 5,
    "marketList": [
      {
        "marketAddress": "0x...",
        "title": "Will BTC hit $100k?",
        "options": [
          { "title": "Yes", "price": "0.65" },
          { "title": "No", "price": "0.35" }
        ]
      }
    ]
  }
}
```
