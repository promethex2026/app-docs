---
title: List Markets
description: "Get paginated list of prediction markets with filters"
---

## Endpoint

```
GET /bayes/market/get-markets
```

**Auth required:** No

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `page` | number | Yes | Page number (1-indexed) |
| `pageSize` | number | Yes | Items per page (max 50) |
| `tag` | string | No | Filter by tag |
| `status` | string | No | Filter by status (`running`, `settling`, `ended`) |
| `sortType` | string | No | Sort order (`hot`, `newest`, `ending_soon`, `volume`) |
| `categoryId` | string | No | Filter by category |
| `hotWords` | string | No | Filter by trending keywords |
| `followed` | boolean | No | Only followed markets (requires auth) |
| `newMarket` | boolean | No | Only recently created |
| `soonExpired` | boolean | No | Ending within 24 hours |
| `baseTokenAddress` | string | No | Filter by base token |
